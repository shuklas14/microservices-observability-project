# Microservices Observability Project

## 🚀 Tech Stack

* Java + Spring Boot
* Microservices Architecture
* Eureka (Service Discovery)
* API Gateway
* JWT Authentication
* Kafka (Event-driven)
* Prometheus + Grafana (Monitoring)
* Docker

## 📊 Features

* Order & Inventory Microservices
* Secure APIs using JWT
* Retry & fault tolerance
* Real-time monitoring dashboard

## 🛠️ How to Run

1. Start Eureka
2. Start services (Order, Inventory, Auth)
3. Start API Gateway
4. Run Prometheus & Grafana via Docker

## 📈 Monitoring

* Prometheus: http://localhost:9090
* Grafana: http://localhost:3000

This project demonstrates production-level backend architecture.
